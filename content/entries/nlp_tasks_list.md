# Tasks in NLP and Computational Linguistics

## Unsupervised

- word definition or translation inference
- loanword / cognate detection, and false cognate detection
- word clustering by lemma or stem

## Supervised

- automatic language detection (written or spoken)
- audio transcription, caption generation
- phonetic transcription of words
- machine translation
- sentiment analysis
- grammatical correctness detection
- translation correctness detection
